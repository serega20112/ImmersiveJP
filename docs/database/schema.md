# Схема базы данных

Проект использует PostgreSQL и Alembic.

Таблицы первого ревижена:

- `users`
  - email, password hash, display name
  - флаг подтверждения почты
  - цель, уровень языка, интересы
  - признак завершенного онбординга
- `learning_cards`
  - пользователь
  - трек: `language`, `culture`, `history`
  - тема, объяснение, примеры, ключевые слова
  - номер партии и позиция внутри партии
- `card_completions`
  - связь пользователя с пройденной карточкой
  - timestamp завершения
- `learning_sessions`
  - текущее состояние по треку
  - номер последней сгенерированной партии

Redis используется отдельно от PostgreSQL:

- кэш ответов LLM;
- rate limiting генерации;
- коды подтверждения почты;
- blacklist JWT.
